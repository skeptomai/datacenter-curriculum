### Critical Interactions

**1. VM to VM Communication (Same Rack):**
```
VM1 ‚Üí virtio ‚Üí vhost ‚Üí NIC ‚Üí Leaf ‚Üí NIC ‚Üí vhost ‚Üí virtio ‚Üí VM2
                       ‚îî‚îÄ VXLAN encapsulated if different VNIs
```

**2. VM to VM Communication (Different Racks):**
```
VM1 ‚Üí virtio ‚Üí NIC ‚Üí Leaf1 ‚Üí Spine ‚Üí Leaf2 ‚Üí NIC ‚Üí virtio ‚Üí VM2
      ‚îî‚îÄ ECMP picks path based on 5-tuple hash
      ‚îî‚îÄ VXLAN overlay provides L2 connectivity
```

**3. High-Performance Path (SR-IOV):**
```
VM ‚Üí SR-IOV VF ‚Üí NIC ‚Üí Network (DIRECT, zero exits!)
     ‚îî‚îÄ IOMMU ensures isolation
     ‚îî‚îÄ Near bare-metal performance
```

**4. Storage over RDMA:**
```
VM ‚Üí virtio-blk ‚Üí vhost-blk ‚Üí RDMA NIC ‚Üí Network
                   ‚îî‚îÄ Zero-copy transfer
                   ‚îî‚îÄ PFC ensures losslessness
                   ‚îî‚îÄ <2Œºs latency
```
