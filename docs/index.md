## Curriculum Structure

```
Foundations          → Core concepts everyone needs
Intermediate         → Networking deep-dives, RDMA, complete virtualization
Containers           → Fundamentals through Kubernetes security
Specialized          → Pick based on your role
Reference            → Acronyms, decision frameworks, setup guides
```

---

## Topics Covered

**Virtualization** — VT-x, EPT, VM exits, device passthrough, Firecracker  
**Networking** — VLAN/VXLAN, EVPN/BGP, Geneve, OVS, Cilium, eBPF, service mesh  
**Storage** — RDMA, NVMe-oF, PFC/DCB, container volumes  
**Containers** — cgroups, namespaces, OCI runtimes, Kubernetes, pod security
